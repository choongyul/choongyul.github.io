프로그래밍 언어론 - Day1

September 23, 2020 in TIL, Programming
프로그래밍 언어란? 자동 실행을 염두에 두고 만들어진 언어다. 프로그래밍 언어 자체는 논리학과 수학 아래서 발전할 수 있다. 예제> 1. 컴퓨터라는 기계에 명령하는 도구이다. 2. 프로그래머들 사이의 소통 수단이다. 3. 상위 레벨 디자인을 표현하는 도구이다. 4. 알고리즘을 기술하는 표기법이다. 소프트웨어는 미개하다. 제대로 동작하는지 검증할 수 없다. 다리를 건설한다고 하면, 물리학과 수학을 통해 안전과 최대 중량 등을 검토할 수 있다. 하지만 소프트웨어는 어느 상황에서 어떤 에러가 발생하는지 미리 확인할 수 없다.
